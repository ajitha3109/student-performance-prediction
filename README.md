# 🎓 Student Performance Prediction

## 📊 Overview
This project uses Machine Learning to predict student exam scores based on various factors such as study hours, class attendance, and sleep duration.

## 🧠 Problem Statement
Understanding student performance is important for improving learning outcomes. This project aims to analyze key factors affecting exam scores and build a predictive model.

## 📁 Dataset
The dataset includes:
- Study Hours
- Class Attendance
- Sleep Hours
- Exam Score

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 🤖 Model Used
- Linear Regression

## 📈 Results
The model shows a positive correlation between study hours and exam scores.  
Multiple factors contribute to performance, and the model predicts results with reasonable accuracy.

## 📊 Visualizations
- Study Hours vs Exam Score
- Actual vs Predicted Scores

## 🚀 How to Run
1. Open the notebook in Google Colab
2. Upload the dataset
3. Run all cells

## 💡 Future Improvements
- Use advanced models (Random Forest, XGBoost)
- Include more features for better accuracy
- Hyperparameter tuning

## 🙌 Conclusion
This project demonstrates how Machine Learning can be used to analyze and predict student performance based on real-world factors.
